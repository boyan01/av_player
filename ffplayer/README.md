# ffplayer

player warpper for ffmpeg.

## compile & Run

### Linux (Ubuntu 20.04)

1. Install av libs.
```shell
sudo apt install libavcodec-dev libavformat-dev libavdevice-dev
```
2. Install sdl2
```shell
sudo apt-get install libsdl2-dev
```

3. Open the project (`audio_player/ffplayer`) in clion/vscode, click **run**.
